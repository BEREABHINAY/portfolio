DocuMind — RAG-Based Document Q&A System

A Retrieval-Augmented Generation system that lets you upload PDFs or text files and ask questions about them in plain English, answered strictly from the document content — with the exact source passages shown alongside every answer, and a custom evaluation harness to measure retrieval quality.

  • Built a full RAG pipeline: chunking, local sentence-transformers embeddings, Qdrant vector search, and grounded generation via Llama 3.1 (Groq), with a system prompt engineered to prevent hallucination.
  • Designed and ran a retrieval evaluation harness measuring Source Hit Rate@K and Keyword Coverage@K against a custom question set, achieving 100% on both metrics.
  • Implemented session-scoped multi-tenant document isolation at the vector-database level, verified with dedicated automated tests, so concurrent users can never see each other's uploaded documents.
